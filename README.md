# Assignment 1: E-Commerce Sales Database Analysis

## Objective

Analyze e-commerce sales data using SQL concepts such as:

* SELECT
* WHERE
* GROUP BY
* HAVING
* JOINS
* CASE Statements
* Transactions
* Constraints
* Indexes

## Database Tables

### Customers

Stores customer information.

### Products

Stores product details and pricing information.

### Orders

Stores customer order information.

### Order Items

Stores products associated with each order.

## Tasks Performed

### Data Retrieval

* Retrieved all customer records.
* Retrieved selected customer information.
* Retrieved unique product categories.

### Filtering

* Delivered orders analysis.
* Electronics products above ₹2000.
* Maharashtra customers joined in 2024.
* Orders within specific date ranges.

### Aggregation

* Total orders count.
* Revenue analysis.
* Category-wise average prices.
* Revenue by order status.
* Product price comparisons.

### Joins

* Customer order analysis.
* Product order analysis.
* Multi-table relationship analysis.

### Advanced SQL

* CASE statements for price tiers.
* Delivered vs Not Delivered analysis.
* Transaction management.
* Foreign key validation.

## Key Insights

1. Delivered orders generated the highest revenue.
2. Electronics products contributed significantly to sales.
3. Some customers placed multiple orders.
4. Foreign key constraints maintain data integrity.
5. Aggregation helps identify high-performing categories.
